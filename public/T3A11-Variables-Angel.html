    int - stores integers (whole numbers), without decimals such as 123 or -123
    double - stores floating point numbers, with decimals, such as 19.99 or -19.99
    char - stores single characters, such as 'a' or 'b', Char values are surrounded by double quotes
    string - stores text, such as "hello world" or "chess". String values are surrounded by double quotes
    bool - stores values with two states: true or false? (true)
